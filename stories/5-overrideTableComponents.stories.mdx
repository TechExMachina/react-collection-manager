# Specific form

You can pass a custom component to override components in material-table

<Preview>
  <Story name="Standard">
    <ReactCollectionManager
      title="Example"
      columns={columns}
      entries={entries}
      canAdd={true}
      schema={schema}
      tableComponents={{
        Action: (props) => {
          //If isn't the add action
          console.log('props.action.tooltip:', props.action)
          if (typeof props.action === typeof Function || props.action.tooltip !== 'Add') {
            return <MTableAction {...props} />
          } else {
            return <button onClick={props.action.onClick}>Add</button>
          }
        },
      }}
    />
  </Story>
</Preview>
